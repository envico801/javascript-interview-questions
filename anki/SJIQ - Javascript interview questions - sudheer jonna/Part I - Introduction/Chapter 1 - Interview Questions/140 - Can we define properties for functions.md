========== Question ==========  

### Can we define properties for functions  

========== Answer ==========  

Yes, We can define properties for functions because functions are also objects.

```javascript
fn = function (x) {
    //Function code goes here
};

fn.name = 'John';

fn.profile = function (y) {
    //Profile code goes here
};
```
