========== Question ==========  

### Is it possible to debug HTML elements in console  

========== Answer ==========  

Yes, it is possible to get and debug HTML elements in the console just like
inspecting elements.

```javascript
const element = document.getElementsByTagName('body')[0];
console.log(element);
```
