========== Question ==========  

### What are primitive data types  

========== Answer ==========  

A primitive data type is data that has a primitive value (which has no
properties or methods). There are 7 types of primitive data types.

1. string

2. number

3. boolean

4. null

5. undefined

6. bigint

7. symbol
