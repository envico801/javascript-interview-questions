==================== Question ====================  

### How do you create an array with some data  

==================== Answer ====================  

You can create an array with some data or an array with the same values using
`fill` method.

```javascript
var newArray = new Array(5).fill('0');
console.log(newArray); // ["0", "0", "0", "0", "0"]
```
