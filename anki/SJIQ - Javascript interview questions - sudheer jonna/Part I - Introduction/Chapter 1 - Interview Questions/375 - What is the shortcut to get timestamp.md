==================== Question ====================  

### What is the shortcut to get timestamp  

==================== Answer ====================  

You can use `new Date().getTime()` to get the current timestamp. There is an
alternative shortcut to get the value.

```javascript
console.log(+new Date());
console.log(Date.now());
```
