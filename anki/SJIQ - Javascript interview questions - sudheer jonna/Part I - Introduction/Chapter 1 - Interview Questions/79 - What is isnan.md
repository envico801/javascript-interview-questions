==================== Question ====================  

### What is isNaN  

==================== Answer ====================  

The isNaN() function is used to determine whether a value is an illegal number
(Not-a-Number) or not. i.e, This function returns true if the value equates to
NaN. Otherwise it returns false.

```javascript
isNaN('Hello'); //true
isNaN('100'); //false
```
