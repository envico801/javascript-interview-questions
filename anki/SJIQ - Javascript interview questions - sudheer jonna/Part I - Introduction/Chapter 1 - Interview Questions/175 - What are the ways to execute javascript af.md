==================== Question ====================  

### What are the ways to execute javascript after page load  

==================== Answer ====================  

You can execute javascript after page load in many different ways,

1. **window.onload:**

```javascript
window.onload = function ...
```

1. **document.onload:**

```javascript
document.onload = function ...
```

1. **body onload:**

```javascript
<body onload="script();">
```
