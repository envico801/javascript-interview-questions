========== Question ==========  

### How do you get unique values of an array  

========== Answer ==========  

You can get unique values of an array with the combination of `Set` and rest
expression/spread(...) syntax.

```javascript
console.log([...new Set([1, 2, 4, 4, 3])]); // [1, 2, 4, 3]
```
