==================== Question ====================  

### What is eval  

==================== Answer ====================  

The eval() function evaluates JavaScript code represented as a string. The
string can be a JavaScript expression, variable, statement, or sequence of
statements.

```javascript
console.log(eval('1 + 2')); //  3
```
