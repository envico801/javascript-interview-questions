==================== Question ====================  

### What are the syntax rules of JSON  

==================== Answer ====================  

Below are the list of syntax rules of JSON

1. The data is in name/value pairs
2. The data is separated by commas
3. Curly braces hold objects
4. Square brackets hold arrays
