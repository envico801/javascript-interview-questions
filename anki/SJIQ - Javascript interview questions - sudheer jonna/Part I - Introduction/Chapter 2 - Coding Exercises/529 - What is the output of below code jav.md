==================== Question ====================  

### What is the output of below code

```javascript
let arr = [1, 2, 3, 4, 5, -6, 7];
arr.length = 0;
console.log(arr);
```

- 1: 0
- 2: Undefined
- 3: null
- 4: [ ]  

==================== Answer ====================  

##### Answer: 4

The length of the array 'arr' has been set to 0, so the array becomes empty.
