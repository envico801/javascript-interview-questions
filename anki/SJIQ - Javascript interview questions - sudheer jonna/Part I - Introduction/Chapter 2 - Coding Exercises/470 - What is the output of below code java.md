==================== Question ====================  

### What is the output of below code

```javascript
console.log(NaN === NaN);
```

- 1: True
- 2: False  

==================== Answer ====================  

##### Answer: 2

JavaScript follows IEEE 754 spec standards. As per this spec, NaNs are never
equal for floating-point numbers.
