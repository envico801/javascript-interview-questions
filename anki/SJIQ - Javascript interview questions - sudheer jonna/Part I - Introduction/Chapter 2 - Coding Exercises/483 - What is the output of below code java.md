==================== Question ====================  

### What is the output of below code

```javascript
const [x, ...y, z] = [1, 2, 3, 4];
console.log(x, y, z);
```

- 1: 1, [2, 3], 4
- 2: 1, [2, 3, 4], undefined
- 3: 1, [2], 3
- 4: SyntaxError  

==================== Answer ====================  

Answer: 4  
It throws a syntax error because the rest element should not have a trailing
comma. You should always consider using a rest operator as the last element.
